Игра Asteroids

Управление:
Клавиши направления, wad – движение корабля. Возможно только движение вперед либо поворот вправо/влево.
Клавиша пробел – выстрел.

На каждом уровне создается некоторое число астероидов, которые необходимо уничтожить с помощью выстрела. Астероиды отличаются по размерам. При успешном попадании в астероид большего размера создаются два меньших до тех пор, пока не достигается минимальный возможный размер. Астероид минимального размера при попадании уничтожается. 
При каждом попадании увеличивается количество заработанных очков.
При столкновении с астероидом игрок теряет одну жизнь. Спустя некоторое количество времени игрок может продолжить игру, если количество жизней >0, иначе игра прекращается.
 Также в сцене присутствует вражеский корабль, который пытается уничтожить игрока с помощью собственного выстрела. При успешном попадании игрок также теряет жизнь. Корабль можно уничтожить, но через какое-то время он появляется снова.

SpaceshipControls.cs описывает поведение управляемого игроком корабля.
AsteroidScript.cs – поведение астероидов.
AlienScript.cs – вражеского корабля.
GameManagerScript.cs – управление некоторыми элементами игры.
